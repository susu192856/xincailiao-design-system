## 快速开始

- 安装：`npm install`
- 开发：`npm run dev`
- 构建：`npm run build`
- 验证：`npm run verify`

## 项目定位

这是新材道设计系统（xincailiao-design-system）的工作仓库，用于持续维护设计规范、网页文档、设计 Token、组件规范和 Figma 组件库同步资产。

详见 [README.md](./README.md)
