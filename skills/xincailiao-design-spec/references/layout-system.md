# 布局系统 (Layout System)

覆盖官网门户和后台两种骨架模式的排版、网格和响应式规则。

---

## 两种骨架

| 属性 | 官网 Portal | 后台 Dashboard |
|---|---|---|
| 导航 | 顶部导航 64px | 顶栏 64px + 侧栏 240px |
| 内容宽度 | 最大 1400px（居中） | 自适应剩余宽度 |
| 背景 | 白色为主 | `--neutral-50` 全页 |
| 主色 | `--brand-*` | `--product-blue-*` |
| 滚动 | 整页滚动 | 内容区独立滚动 |
| 典型宽度 | 1920px 设计 | 1440px 设计 |

## 官网内容区网格

```
12 列网格，列间距 --spacing-lg (24px)

│ ← --spacing-4xl → │  内容  │ ← --spacing-4xl → │
│      (96px)       │ 1400px │      (96px)       │
```

- 卡片行：3列或4列排列
- 文字行：居中 max-w-[720px]
- 全宽图片/视频：max-w-[1400px]

## 后台内容区

- 侧栏展开时内容区宽度 = 屏幕宽度 - 240px
- 侧栏折叠时内容区宽度 = 屏幕宽度 - 56px
- 内容区内边距固定 `--spacing-lg` (24px)
- 表格横向溢出时滚动，不压缩列

## Z-index 层级

```
--z-tooltip: 600    ← 最顶层（tooltip 必须可见）
--z-toast:   500
--z-modal:   400
--z-overlay: 300    ← 遮罩（抽屉/侧栏 overlay）
--z-sticky:  200    ← 表头固定、顶栏
--z-dropdown: 100   ← 下拉菜单
```

严格遵守此层级，不自定义 z-index。

## 间距使用指引

| 间距 | 用途 |
|---|---|
| `xs` (4px) | 图标与文字、tag 内边距 |
| `sm` (8px) | 表单标签与输入框、选项之间 |
| `md` (16px) | 组件之间、表格单元格内边距、段落间距 |
| `lg` (24px) | 内容区 padding、卡片之间、标题到内容 |
| `xl` (32px) | 区块之间（小） |
| `2xl` (48px) | 区块之间（中）、footer padding |
| `3xl` (64px) | 内容区之间 |
| `4xl` (96px) | 大区块间隔、Hero 上下 |

## 两种模板何时使用

| 页面类型 | 使用模板 |
|---|---|
| 官网首页、产品介绍、解决方案、关于我们 | Portal |
| 登录、注册、忘记密码 | Portal（简化版，无导航无footer） |
| 后台首页 Dashboard | Dashboard |
| 列表管理（用户、订单、商品...） | Dashboard |
| 新增/编辑表单 | Dashboard |
| 详情查看 | Dashboard |
| 数据看板/图表 | Dashboard |
| 系统设置/权限配置 | Dashboard |
| 材库前端展示 | Portal |
| 材库后台管理 | Dashboard |

## 多产品线布局差异

各产品线的布局差异仅在以下维度变化（见 `assets/product-lines.json`）：

- 主色调（品牌色 vs 产品蓝 vs 自定义）
- 导航高度（标准 64px）
- 侧栏宽度（标准 240px）
- 内容区最大宽度
- 字体缩放（部分产品线需要更大字号）

**布局骨架本身不变** — 不因产品线不同而改变页面结构。
