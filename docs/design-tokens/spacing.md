# 间距

> 新材道设计系统 — 间距 Token

## Token 列表

| Token | 值 | 说明 |
|------|-----|------|
| `--spacing-xs` | `4px` | 极小间距 |
| `--spacing-sm` | `8px` | 小间距 |
| `--spacing-md` | `16px` | 中间距 |
| `--spacing-lg` | `24px` | 大间距 |
| `--spacing-xl` | `32px` | 较大间距 |
| `--spacing-2xl` | `48px` | 大间距 |
| `--spacing-3xl` | `64px` | 很大间距 |
| `--spacing-4xl` | `96px` | 极大间距 |

## CSS 使用方式

```css
/* 直接在样式文件中引用 */
.element {
  gap: var(--spacing-xs);
}
```

## 间距使用原则

- **xs (4px)** — 图标与文字极小间距
- **sm (8px)** — 按钮内部元素 / 标签内边距
- **md (16px)** — 表单项间距 / 卡片内边距
- **lg (24px)** — 模块内部区块间距
- **xl (32px)** — 页面区块常规间距
- **2xl (48px)** — 大模块分隔
- **3xl-4xl (64-96px)** — 官网大区块间距

## 场景配方

| 场景 | 建议间距 | 说明 |
|------|----------|------|
| 后台表单字段之间 | 16px | 保持可扫读且不松散 |
| 后台表单分组之间 | 24px | 用于区分配置区块 |
| 底部按钮组 | 12px | 取消和确认按钮之间的常规距离 |
| 筛选项之间 | 12px | 适合高密度筛选区域 |
| 工具栏到表格 | 12px-16px | 保持操作区与数据区贴近 |
| 表格到分页 | 16px | 保持分页从属于表格 |
| 官网标题到正文 | 16px-24px | 保证叙事节奏 |
| 官网大区块之间 | 64px-96px | 用于首页、门户和专题页 |
