# Z-index

> 新材道设计系统 — z-index Token

## Token 列表

| Token | 值 | 说明 |
|------|-----|------|
| `--z-dropdown` | `100` | 下拉菜单 |
| `--z-sticky` | `200` | 粘性定位 |
| `--z-overlay` | `300` | 遮罩层 |
| `--z-modal` | `400` | 弹窗 |
| `--z-toast` | `500` | 消息提示 |
| `--z-tooltip` | `600` | 工具提示 |

## CSS 使用方式

```css
/* 直接在样式文件中引用 */
.element {
  z-index: var(--z-dropdown);
}
```

## 层级说明

| Token | 值 | 适用场景 |
|------|-----|----------|
| `--z-dropdown` | `100` | 下拉菜单 |
| `--z-sticky` | `200` | 粘性定位 |
| `--z-overlay` | `300` | 遮罩层 |
| `--z-modal` | `400` | 弹窗 |
| `--z-toast` | `500` | 消息提示 |
| `--z-tooltip` | `600` | 工具提示 |

> 层级间隔为 100，方便未来在中间插入新层级而不冲突。
