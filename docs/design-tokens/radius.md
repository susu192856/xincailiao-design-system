# 圆角

> 新材道设计系统 — 圆角 Token

## Token 列表

| Token | 值 | 说明 |
|------|-----|------|
| `--radius-none` | `0px` | 表格/数据网格 |
| `--radius-sm` | `2px` | 交互控件（按钮/输入框等） |
| `--radius-md` | `4px` | 页面标题背景/标准卡片/提示与分组容器 |
| `--radius-lg` | `8px` | 品牌展示大卡片/独立浮层/对话框外壳 |
| `--radius-xl` | `12px` | 特殊官网视觉/媒体蒙版 |
| `--radius-2xl` | `16px` | 特殊品牌视觉/媒体蒙版 |
| `--radius-full` | `9999px` | 胶囊/头像/圆形 |

## CSS 使用方式

```css
/* 直接在样式文件中引用 */
.element {
  border-radius: var(--radius-none);
}
```

## 圆角使用原则

| 层级 | 圆角 | 适用场景 |
|------|------|----------|
| radius-none | 0px | 表格、数据网格、强秩序信息区 |
| radius-sm | 2px | 按钮、输入框、选择器、标签等高频交互控件 |
| radius-md | 4px | 页面标题背景、标准内容卡片、提示说明区、代码区和分组容器 |
| radius-lg | 8px | 品牌展示大卡片、独立浮层、对话框外壳 |
| radius-xl | 12px | 特殊官网视觉或媒体蒙版，不用于重复内容卡片 |
| radius-2xl | 16px | 特殊品牌视觉或媒体蒙版，不用于后台 |
| radius-full | 9999px | 胶囊标签、头像、圆形按钮 |
